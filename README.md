# soledo-static

Static personal finance tracker with multi-currency display, local storage persistence, and JSON import/export.

## Structure

- `index.html` contains the app markup.
- `assets/css/styles.css` contains the UI styling.
- `assets/js/app.js` contains the finance tracker logic and exchange-rate handling.
- `assets/js/currency-picker.js` contains the custom currency dropdown behavior and symbol normalization.

## Run

Open [index.html](./index.html) in a browser, or serve the folder with any static file server.
